# CP164 – Data Structures I

**Wilfrid Laurier University**  
**Language:** Python 3 | **Tools:** Eclipse / PyDev

---

## About This Course

CP164 is an intermediate programming course focused on the design and implementation of fundamental data structures and algorithms. Topics covered include:

- Object-oriented programming (classes, objects, attributes, methods)
- Stacks and queues (array-based and linked implementations)
- Linked lists and sorted lists
- Recursion and recursive algorithms
- Searching and sorting algorithms
- Heaps and priority queues
- Binary Search Trees (BSTs)
- Algorithm analysis and Big O notation
- Sets and hashing

All code is written in Python 3 and follows the CP164 coding and documentation standards, including structured docstrings, PEP8 style, and descriptive naming conventions.
